# Test Strategy Socialization Plan

## Purpose
Promote shared ownership, best practices, and empathy for testing across teams.

## Initiatives
- **Workshops:** Monthly hands-on sessions on TDD, test writing, and feedback loops
- **Bootcamps:** Quarterly deep-dives for new joiners and interested engineers
- **Pairing:** Encourage pairing with experienced test writers
- **Empathy Building:** Include time for tests in engineering estimates; highlight time saved by tests
- **Best Practices:** Document and share cross-team learnings

## Action Items
- Schedule first workshop by 2025-10-01
- Add test empathy to sprint planning
- Share best practices in EM forums
